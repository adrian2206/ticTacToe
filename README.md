**Tic-Tac-Toe: Module Pattern Implementation**
  A browser-based Tic-Tac-Toe game built with Vanilla JavaScript, focusing on clean code architecture and the Module Pattern.
  The primary goal of this project was to minimize global code by encapsulating logic within Factory Functions and IIFEs (Immediately Invoked Function Expressions).

**Key Features**
  Encapsulated Logic: Separate modules for the Gameboard, Display Controller, and Game Flow.
  Factory Functions: Used to dynamically create player objects.
  Smart UI: Real-time DOM updates, player name customization, and a result display with a restart option.
  Pure JS: No global variables; all functionality is tucked into logical, reusable objects.
